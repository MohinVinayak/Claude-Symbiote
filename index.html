System Design
Three-layer architecture
🌐

Browser Extension

MV3 content script watches the claude.ai DOM via MutationObserver and intercepts the SSE fetch stream to fire precise chunk timestamps.

Chrome MV3 Firefox Arc
🔗

WebSocket Bridge

Background service worker maintains a persistent WebSocket to ws://127.0.0.1:7429 with exponential backoff reconnect on drop.

tokio-tungstenite JSON events
🖥️

Desktop Overlay

Tauri 2 transparent, always-on-top window. Three.js GLSL fragment shader renders the liquid glass capsule. OS APIs track the Claude window position.
